Comparison of Cellranger V2 against V3 with references generated with V2 and/or V2.

01-mkref

	Directory containing reference generation for v2 and v3, generated from an identical set
	of input files (download script in reffiles). Folders v2 and v3 contain the relevant
	reference generation.

02-count

	Contains the cellranger runs for the V2 and V3. V2 has also been run with a V2 generated
	reference. The name 'execV3_annotV2' refers to a run with V3 but annotation generation with V2.

03-examine

	The R scripts used to produce the figures and analysis for the reports.
